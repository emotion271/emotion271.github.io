PAT A1047 Student List for Course

发表于 2019-03-30 | 阅读次数:

题目：A1047 Student List for Course

题意：

  给出$N$个学生的选课信息，第$i$个学生所选的课有$C$门，后接所选课的课程编号。要求输出每门课所选的所有学生的姓名。要求课程编号升序，学生姓名按字典序。

思路：

  最容易想到的是用$map\langle int,set\langle string \rangle\rangle$来存储，但由于常数大，这样写会超时。所以就想到用$map\langle int,vector\langle int \rangle\rangle$来存储，即对学生姓名进行$hash$，用$vector$来存储，只要在输出的时候排序后再还原成学生姓名就可以了。我这样写$AC$了。但是其实这题的卡时间其实在输入输出上，将$string$换成$char$来存储就可以了，那就不用$cout$来输出输出了，也没必要进行$hash$。
