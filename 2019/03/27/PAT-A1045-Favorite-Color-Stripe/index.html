PAT A1045 Favorite Color Stripe

发表于 2019-03-27 | 阅读次数:

题目：A 1045 Favorite Color Stripe

题意：

  总共给出$N$种颜色，给出一个长度为$M$的颜色序列$S_1$，这个序列是按照小女孩喜欢的颜色排的。给出一个长度为$L$的目标序列$S_2$，求序列$S_2$中按序列$S_1$顺序排列的序列的最长的长度，序列$S_1$中的颜色不一定都要出现。

思路1：

  看完题目后最直接想到的是用最长公共子序列$(LCS)$，但是本题中序列$S_2$中连续多次出现的元素是可以与$S_1$中的一个元素进行匹配的。

  首先考虑$LCS$的动态规划方程

  对于本题，当$s_1[i]==s_2[j]$时，当$s_2$中没有出现连续元素时，状态转移和$LCS$的相同；当出现连续元素时，状态的转移可以来自$dp[i][j-1]$.所以动态规划方程可以写成

思路2：

  这题还可以用最长上升子序列$（LIS）$来解。首先可以将序列$S_1$中的元素$hash$成对应的位置信息，如样例中给出的序列$\{2,3,1,5,6\}$hash后为$\{3,1,2,0,4,5,0\cdots\}$。然后将序列$S_2$中的元素也进行hash，并舍去$hash$值为$0$的元素，如样例中的序列$\{2,2,4,1,5,5,6,3,1,1,5,6\}$$hash$去$0$后为$\{1,1,3,4,4,5,2,3,3,4,5\}$。最后只需求该序列的最长上升子序列的长度就可以了。

$LIS$的求法:

$O(n^2)​$的$dp​$:

  $dp[i]$表示以第$i$个元素结尾的最长上升子序列的长度

$O(nlogn)$的贪心：

  用数组$b[i]$来存长度为$i$的最长上升子序列的结尾元素的最小值，那么我们只要维护数组$b[i]$就可以了。维护$b[i]$使其结尾元素尽可能小，这样才能使上升子序列尽可能的长。
