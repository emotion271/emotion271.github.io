PAT A1050 String Subtraction

发表于 2019-03-31 | 阅读次数:

题目：A1050 String Subtraction

题意：

  将出现在$S_2$中的字符从$S_1$中删除，输出处理后的$S_1$。

思路：

  这题要求我们复杂度尽可能低，所以每次都进行查询肯定是不可取的。那么我们可以将这个查询的过程用$hash$变成$O(1)$,即对$S_2$中字符进行$hash$,$0$表示字符未在$S_2$出现，$1$表示出现。

  之后我试了一下用$string.find()$来查找也是可以$AC$的。
